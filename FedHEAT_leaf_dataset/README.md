# Experimental Evaluation for FEMNIST and CelebA

This directory contains the experimental code for the FEMNIST and CelebA datasets.  
All datasets are generated and preprocessed using the LEAF federated learning benchmark library[5](https://github.com/TalwalkarLab/leaf).

## Dataset

1. Follow the dataset download and preprocessing procedure provided by the LEAF benchmark library.
In our experiments, preprocessing was performed using the command below.
~~~bash
./preprocess.sh   -s niid   --sf 1.0   -k 10   -t sample   --tf 0.9   -smplseed 1549786796   -spltseed 1549786595
~~~

2. Under the `./data/` directory, create the following folders and move the corresponding `train/` and `test/` directories generated by the LEAF library into them:
- `leaf_femnist/`
- `leaf_celeba/`

3. For the CelebA dataset, create the `data/raw/` directory inside `leaf_celeba/` and place the following raw files into it:
- `img_align_celeba/`
- `identity_CelebA.txt`
- `list_attr_celeba.txt`

## Training
The commands below can be used to evaluate the federated learning methods.  
Users can freely specify the `exp_name` argument, while the recommended hyper-parameter settings for each dataset are provided below.

The `train.encrypted` option determines whether the experiment is performed in the plaintext or encrypted setting.

To select a dataset, set the `dataset` argument to one of the following:
- `femnist`
- `celeba`

For example, the following command runs the evaluation on FEMNIST:
**FEMNIST, FedHEAT in encrypted version:**
~~~bash
CUDA_VISIBLE_DEVICES=0 python3 \
    federated_train.py \
	multiprocessing=True \
	main_gpu=0 \
	exp_name=FedAdam_ct_proposed_tau-in_10Minus5_encL3 \
	trainer=leaf \
    client=base \
    dataset=leaf_femnist \
    model=leaf_femnist_cnn \
    output_model_path=cnn-2layer.pt \
    server=FedAdam \
	trainer.encrypted=True \
    server.algo=proposed \
	server.tau_in=True \
	server.tau=0.00001 \
	trainer.enc_level=3 \
    server.t1=100 \
	server.t2=90 \
    split.alpha=0.3 \
	batch_size=20 \
	trainer.local_lr=0.1 \
	trainer.local_epochs=1 \
	trainer.global_lr=0.01 \
	trainer.global_rounds=500 \
    wandb=True
~~~

## Hyper-parameters
### Table. Hyperparameters for FEMNIST and CelebA. $\tau$ denotes the stabilization parameter; $t_1$ and $t_2$ denote moment thresholds; $T$, $K$, $\eta$, and $\eta_l$ denote the total number of rounds, local steps, server learning rate, and client learning rate, respectively.

| Hyperparameter | FEMNIST | CelebA |
|----------------|---------|---------|
| $\tau$         | $10^{-5}$ | $10^{-4}$ |
| $t_1$          | 100     | 100     |
| $t_2$          | 90      | 90      |
| $T$            | 500     | 500     |
| $K$            | 1       | 1       |
| $\eta$         | 0.01    | 0.01    |
| $\eta_l$       | 0.1     | 0.1     |
| Batch size     | 20      | 20      |
